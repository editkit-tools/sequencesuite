Release notes

SequenceSuite 1.2.0

A big pass on speed and everyday handling — sequences open faster, and the report tables now feel like a proper macOS list you can drive entirely from the keyboard.

Faster on real timelines

  • Quicker opens — repeated reads of the same source are reused instead of re-scanned, so loading a sequence and switching between Sheet, Gather and Dupe is noticeably faster.
  • No more stall on mode switch — moving between modes on a heavy timeline no longer hangs the window.

Drive the tables from the keyboard

  • Full keyboard navigation — arrow keys, plus Home, End, Page Up and Page Down to move through long clip lists without reaching for the mouse.
  • Multi-row selection — Shift- and ⌘-click to pick ranges or scattered rows, and ⌘A to select everything.
  • Selection totals — select a span of clips and Timecode shows the running total for just that selection.
  • Copy what you picked — the Copy command now reflects how many rows you have selected and copies exactly those.
  • ⌘⌫ to remove, plus right-click Remove in the sidebar to drop a loaded source.
  • Keyboard cheat sheet — a quick reference for every shortcut, and a Recents list to reopen what you were just working on.

Polish

  • Cleaner selection highlighting and stronger contrast in Dark Mode.
  • Accessibility improvements throughout the report tables.
  • Various smaller fixes to row handling and scrolling.
