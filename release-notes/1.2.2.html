Release notes

SequenceSuite 1.2.2

A workflow polish release for moving between Sheet, Gather, and Dupe without losing your place.

Across the suite

  • Right-click handoffs between modes — find duplicates for a Sheet clip, gather a sequence's media, gather every use in a duplicate group, jump from Dupe back to the cut, or show the Sheet clips that use a Gather file.
  • Sheet summary drill-downs — Reel Summary and File Summary rows can now jump straight to the matching clips, with exact matching so similar source names do not get pulled in by mistake.
  • More useful Gather rows — locate missing media, exclude rows from a Gather plan, re-check a file after fixing it in Finder, and copy a selected file list for handoff.

Fixes

  • Dupe target handling is clearer for grouped EDL cuts, with a dismissible target chip.
  • Plain icon buttons have wider hit areas, and held-arrow table navigation stays smoother.
