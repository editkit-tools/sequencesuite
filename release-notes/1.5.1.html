Release notes

SequenceSuite 1.5.1

More control over the poster frames on your exports - choose a playout per timeline, set one by hand, or leave thumbnails off entirely.

Thumbnails you choose, per timeline

  • Export thumbnails now start switched off, so a report never waits on a video it cannot find. Turn them on whenever you want them.
  • When a matching playout sits beside your sequence, SequenceSuite uses it straight away, with no confirmation to click through.
  • A single file holding several timelines can now point each one at its own playout, so every timeline shows the right frame instead of sharing one.

Link a playout by hand

  • Right-click any timeline in the sidebar to Link, Change or Clear the playout used for its thumbnails, without waiting to be asked at export time.

A clearer locate panel

  • When SequenceSuite does ask for a playout, the panel names the timeline and the file it came from in plain terms, and the explanation no longer gets clipped.
  • A new Skip Thumbnails button skips just the current timeline while later ones in the same batch still get asked. Cancel stops asking for the rest of the run.
