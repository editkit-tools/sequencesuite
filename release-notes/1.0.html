Release notes

SequenceSuite 1.0

Sheet reports, media gather, and duplicate detection — three post-production workflows in one Mac app.

SequenceSuite 1.0 brings together the workflows from SequenceSheet Pro, KleptoClip, and DejaView into a single app. Open a timeline file and switch between Sheet, Gather, and Dupe without re-importing. All three modes share the same open files and the same session.

Sheet — sequence reports

Sheet reads your timeline and builds a full clip listing with source reels, in/out timecodes, track kinds, markers, transitions, filters, and graphics. Export to XLSX, CSV, PDF, or ALE in one click.

  • Separate report sections for reels, video, audio, markers, transitions, filters, and graphics
  • Batch export — select multiple sequences and write one file per sequence, or combine into a single workbook
  • Poster-frame thumbnails embedded in XLSX and PDF exports, decoded from an export playout or the original linked media
  • YouTube Chapters export from sequence markers, offset to programme start
  • ALE export for Avid bin import
  • Multi-track EDL sets grouped automatically as one cut

Gather — media consolidation

Gather reads the file paths embedded in your timeline and builds a copy plan. Review what's ready, what's missing, and where duplicates already exist on the destination before a single file moves.

  • Resumable copy with per-file checksum verification
  • Full source folder tree preserved at the destination — volume name and path kept intact for SMB, Nexis, USB, and local drives
  • R3D Siblings — scan a RED camera card against a Resolve conform destination and copy only the missing companions (.rtn, .wav, .rmd, etc.)
  • Gather completion notifications via macOS banners with Show in Finder
  • AAF ingest — reads Avid Media Composer sequence exports and builds a Gather plan from the embedded media locators

Dupe — duplicate detection

Dupe compares timelines to find clips used more than once — across episodes, within a single cut, or both. Results are grouped by source clip with exact, partial, and proximity match classifications.

  • Cross-episode comparison across any combination of open sequences
  • Target Episode mode — anchor the report to one episode's timeline perspective
  • Ignored sources — right-click a row to suppress known intentional reuses
  • Display Names — relabel any EDL or XML sequence without renaming the file
  • PDF View (single target) and PDF Master (all episodes, one document) exports
  • Matrix view showing overlap counts across every episode pair

Supported formats

  • Premiere Pro XMEML — single and multi-sequence XML, including 100+ sequence projects
  • DaVinci Resolve XML — treated as a first-class source alongside Premiere
  • CMX 3600 EDL — including Avid File_16, File_32, and File_129 multi-track exports
  • FCPXML — native 1.12–1.14 packages and EditingTools-converted 1.6–1.10 files; first-pass support for real-world FCP projects
  • OpenTimelineIO — read-only ingest of .otio exports from Resolve and other OTIO adapters
  • Avid AAF — Gather media inventory
