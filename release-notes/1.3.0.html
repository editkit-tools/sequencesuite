Release notes

SequenceSuite 1.3.0

A rebuilt export experience in Sheet - a dedicated Custom Export window, exports that match exactly what you are filtering, and a much faster pass over big multi-sequence projects.

Custom Export window

  • One window for the whole report - rows, columns, sections and packaging in a single place, with a live preview of the file name before you export.
  • Choose how every time value reads - timecode, hours:minutes:seconds, seconds, frames or feet and frames, with independent styles for the sequence and source columns.
  • Save your own export layouts and reuse them. The built-in Standard layout stays untouched.

A clearer Sheet sidebar

  • The sidebar is now pure navigation. A new All Open Files scope lets you browse every open sequence at once.
  • Multi-sequence projects group their clips by sequence, so the order on screen matches the order in the export and cut groups always stay together.

Export what you are looking at

  • Filtered views export exactly the rows you can see - to Excel, CSV, PDF or ALE - with a note recording the subset.
  • Copy Table puts the current table straight on the clipboard, and you can multi-select reels and files to build the exact set you need.

Final Cut metadata

  • Final Cut Pro reports now include Scene, Shot/Take and Camera Angle, read straight from the clip metadata. Columns only appear when the footage actually carries the data, so reports from other editors are unchanged.
  • CSV reports carry the same full column set as Excel, so the two always match.

Fixes and polish

  • Large multi-sequence projects feel snappier to open and browse.
  • PDF reports keep their light, print-ready look whichever appearance the app is in.
  • Clip context-menu copy actions are labelled Sequence TC to match the app's terminology.
