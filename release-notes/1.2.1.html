Release notes

SequenceSuite 1.2.1

A small fix-up release that tightens the keyboard handling and selection totals introduced in 1.2.0.

Fixes

  • Accurate selection totals on mixed-rate timelines — the running total for a selection now adds up at the sequence's frame rate rather than each clip's source rate, so the figure matches your timeline.
  • Arrow keys move the selection again — pressing an arrow key without a modifier now moves the highlighted row instead of stretching the selection.
  • Highlight survives loading another EDL — bringing in a second EDL no longer clears the row you had selected.
