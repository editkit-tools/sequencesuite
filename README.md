# SequenceSuite — Releases

Sparkle update feed and release assets for **SequenceSuite** (macOS).

- **Appcast:** `https://editkit-tools.github.io/sequencesuite/appcast.xml`
- **App:** [editkit.tools/sequencesuite](https://editkit.tools/sequencesuite)

Each release contains `SequenceSuite.dmg`. Release notes in `release-notes/` are served via GitHub Pages and loaded by the Sparkle update dialog.
